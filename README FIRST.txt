1. Execute SQL 
	- 1. Kat localhost, buat database "ppbm"
	- 2. Bukak ppbm.sql guna apa2 text editor/IDE, ctrl A + ctrl C.
	- 3. Kat localhost, bukak tab "SQL", paste content ppbm.sql tadi. 
	- 4. Klik "GO"

2. Buat admin user
	- 1. bukak localhost/newppbm/register.php
	- 2. isi form tu:
		- username: admin,yang lain2 boleh hentam
	- 3. dah register, pergi database, cari user admin tadi.
	- 4. klik edit, dekat column isAdmin,tukar jd 41 (jgn tanya kenapa, haha).
	- 5. dekat column approved, tukar jadi 1.
	- Klik save,setel.
	
3. software testing
	- 1. Register user byk2.
	- 2. Kalau ada apa2 error, bagitau cepat2.

Kalau tak faham apa2 function, or anything, tanya je.
